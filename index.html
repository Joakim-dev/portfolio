This Project

Project Challenges

The challeges I had for this project was:

An all grid-based layout.

A fully functional contactform with PHP.


Solutions

I started with the grid layout, and it actually went surprisingly quick to build the whole page this way. This was just the layout though, I also had to add responsiveness, animations and a functioning navbar. I tried to find if someone else had done the same thing, regarding the navigation inside the one-page, but I could not find any other example.

I built a contact form based on HTML and then added PHP-functions afterward.

When deploying the site I pretty soon discovered that the host didn't allow PHP, so I had to chang my PHP-code to a variant that fitted the host.


Result

The end result is pretty much as I was hoping it would be. Unfortunately, I had to ditch some PHP, but it will come in handy when I change the host.
