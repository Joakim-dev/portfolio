This Project

Project Challenges

The challeges I had for this project was:

An all grid-based layout.

A fully functional contactform with PHP.


Solutions

I started with the grid layout, and it actually went surprisingly quick to build the whole page this way. It was just the layout though.                                 I also had to add responsiveness, animations and a functioning nav-bar. I tried to find if someone else had done the same thing with navigation inside a grid-page, but I couldn't find any.

I built a contact form based on HTML and then I added the PHP-functions afterward.

when deploying the site I pretty soon discovered that the host didn't allow PHP. So I had to change my PHP-code to a variant that fitted the host.


Result

The end result is pretty much as I was hoping it would be. Unfortunately, I had to ditch some PHP, but it will come in handy when I change the host.

My Career

Past

I have done quite a few things in my life, but my main profession before becoming a software-developer was as a doctor of Naprapathy. I ran my own clinic for more than ten years before switching to coding.


The Switch

The main reason for me to switch career was simply a matter of interest. I have always loved to create stuff, to learn things, and to use my mind. I did my first Java-course at a University in Sweden in 2015. I really liked the java-language, but I felt that something was missing.

In 2017 I began with Flutter for mobile applications, where I liked the fact that I could see on a screen what happened and I could easily create a lot of visual stuff.

In 2019 I switched over to web development. It was not that I didn't like Flutter, I just felt it a bit narrow.


Present

Now I can look back at all the things I've done and learned along the way, but I much rather look ahead to all the things I will achieve and learn, both as a human and as a software developer.
